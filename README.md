# mall_manage
商品管理系统


``` bash
#创建vue项目
vue inite webpack name

#启动项目
# 1、初始化项目依赖
npm install 

# 2、启动项目命令
npm install

#项目打包
# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
